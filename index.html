한국어 맞춤법 검사

오픈소스 데스크톱에 널리 사용되는 hunspell 맞춤법 검사 프로그램에서 동작하는 (현재 유일한) 한국어 맞춤법 사전입니다.

현재 여러 리눅스 배포판과 오픈소스 애플리케이션에서 동작합니다.

특징

  • 단어 단위 맞춤법 검사
  • 불규칙 활용을 포함한 용언 활용 등 교착어인 한국어의 특징에 맞게 구현
  • hunspell 맞춤법 검사를 사용하는 프로그램이라면 수정 없이도 한국어 맞춤법 검사가 동작
  • 명확한 오픈소스 라이선스의 단어 데이터만 사용해 리눅스 배포판 및 오픈소스 소프트웨어와 함께 배포 가능

다운로드

Github 다운로드 (소스 및 aff/dic 파일)

리눅스 배포판 및 오픈소스 프로그램에서 손쉽게 사용하려면 다음을 참고하십시오.

  • 데비안: hunspell-ko 패키지 설치, 6.0 (squeeze) 이후
  • 우분투: hunspell-ko 패키지 설치, 9.10 (karmic) 이후
  • 페도라: hunspell-ko 패키지 설치, fc12 이후
  • Mac OS X 10.6 이상: /Library/Spelling 또는 홈폴더/Library/Spelling 아래 aff/dic 파일 복사
  • Mozilla/Firefox 부가 기능
  • LibreOffice 확장
  • Apache OpenOffice 확장

리눅스 배포판의 경우 브라우저나 오피스 프로그램은 보통 내장된 hunspell을 사용하도록 빌드되어 있습니다. 그러니 배포판에 포함된 패키지를 설치하면 프로그램별 확장 기능을 따로 설치하지 않아도 됩니다.
